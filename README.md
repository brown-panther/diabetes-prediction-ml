# Diabetes Prediction using Machine Learning Algorithms with Performance metrics and Holdout method on Egyptian dataset 

## Overview

This reserach presents a machine learning-based approach for predicting diabetes using clinical, demographic, and lifestyle-related data. The study evaluates multiple machine learning algorithms across different data splits to identify the most effective model for diabetes classification.
## Objectives
* To predict diabetes using machine learning techniques
* To compare the performance of multiple models
* To evaluate models using different data split ratios
* To analyze model performance using visualization techniques
  
**##Dataset**
The dataset consists of health-related attributes such as age, glucose level, insulin level, BMI, HbA1c, blood pressure, cholesterol levels, and other clinical indicators.
⚠️ Note: The dataset is not publicly available due to privacy and ethical considerations. Users can upload their dataset manually when running the notebook.
## Data Preprocessing

The following preprocessing steps were applied:

* Handling missing values
* Removing duplicate records
* Encoding categorical variables
* Feature scaling (Standardization)
* Random oversampling to address class imbalance

## Machine Learning Models

The following algorithms were implemented:

* Random Forest (RF)
* Decision Tree (DT)
* Extra Trees (ET)
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* AdaBoost
* Bagging
* Logistic Regression (LR)
* Naïve Bayes (NB)
* Gradient Boosting (GB)

## Evaluation Metrics

Model performance was evaluated using:

* Accuracy
* Precision
* Recall

## Results Visualization

The results are visualized using:

* Performance heatmaps (Accuracy, Precision, Recall)
* Confusion matrix
* Feature importance analysis
